## LaTeX Knowledge & Architecture

This resume demonstrates structured and maintainable LaTeX development.

### Key Features

- Compiled with **XeLaTeX**
- System font integration via `fontspec`
- Custom color scheme and layout design
- Modular document structure using `\input`
- Custom macros for reusable CV entries
- Section styling via `titlesec`
- Clean spacing and typography management
- Icon integration using `fontawesome5`
- Version-controlled document workflow
